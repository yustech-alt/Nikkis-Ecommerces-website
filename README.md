# Nikkis E-commerce

A bold dark-mode marketplace built with React, Tailwind CSS v4, and Vite.

## Tech Stack
- React 18
- React Router DOM v6
- Tailwind CSS v4
- Vite

## Features
- Landing page with hero, categories, featured products, testimonials
- Product listing with filters and sorting
- Product detail page
- Shopping cart with localStorage persistence
- Dark / Light mode toggle
